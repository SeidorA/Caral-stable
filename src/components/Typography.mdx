# Typography

La tipografía de **Caral** prioriza la jerarquía visual y la legibilidad. Utilizamos **Poppins** en todo el espectro del sistema.

## Headings (Display)
Se utilizan para títulos de gran impacto y jerarquías superiores. El peso predeterminado para titulares es **Bold (700)** o **Extrabold (800)**.

## Body & UI Text
Utilizado para el contenido principal, estados de componentes y metadatos. El peso estándar es **Regular (400)**.

## Pesos de Fuente
Poppins soporta múltiples pesos que utilizamos para diferenciar la importancia de la información.

- **Regular (400)**: Texto de cuerpo y párrafos.
- **Medium (500)**: Etiquetas de inputs y metadatos.
- **SemiBold (600)**: Títulos secundarios y botones.
- **Bold (700)**: Títulos principales.
- **ExtraBold (800)**: Títulos de gran tamaño (Display).
